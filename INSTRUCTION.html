üöÄInitial Setup and Installation

1

Download Website Files

Ensure you have received the website files in ZIP format. Extract the files to an easily accessible folder.

üí° Pro Tip: Create a new folder named "coffeeo-landing" to store all files.
2

Install Node.js

This website uses Node.js. Download and install Node.js from the official website:

https://nodejs.org/

Choose the LTS (Long Term Support) version for stability.

3

Install Dependencies

Open Command Prompt/Terminal in the website folder, then run:

npm install

This process will install all required libraries and dependencies.

‚úèÔ∏èContent Modification

1. Changing Coffee Shop Information

Open the index.html file and find the section you want to modify:

Changing Coffee Shop Name:

<div class="text-2xl lg:text-3xl font-bold text-white font-serif">Coffeeo</div>

Replace "Coffeeo" with your coffee shop name.

Changing Address:

<div class="text-sm font-semibold">123 Coffee Street</div>

Replace with your coffee shop address.

Changing Phone Number:

<div class="flex items-center gap-2"><span class="text-coffee-100">üìû</span><span>(021) 1234-5678</span></div>

Replace with the correct phone number.

2. Modifying Coffee Menu

Find the "Coffee Products Section" and modify:

Menu Name:

<h3 class="text-2xl font-bold text-coffee-500 font-serif mb-1">Lungo Coffee</h3>

Menu Price:

<div class="text-2xl font-bold text-coffee-400">$19</div>

Menu Description:

<p class="text-gray-600 text-sm leading-relaxed mb-4">Rich, full-bodied espresso...</p>

3. Replacing Images

To replace images:

  1. Save new images in the public/assets/images/ folder
  2. Update image paths in the index.html file
  3. Supported formats: JPG, PNG, WebP
src="public/assets/images/your-image-name.jpg"
‚ö†Ô∏è Pro Tips: Always backup original files before making changes. Use Ctrl+Z to undo if mistakes occur.

üé®Design Customization

1. Changing Colors

Open the src/style.css file and find the color section:

Main Theme Colors (Coffee Brown):

--coffee-50: #faf8f6;
--coffee-100: #f5f1ed;
--coffee-200: #e8dcc8;
--coffee-300: #d4b895;
--coffee-400: #c19a6b;
--coffee-500: #8B4513;

Change hex color values according to your preferences.

2. Changing Fonts

To change fonts, find the Google Fonts section in index.html:

<link href="https://fonts.googleapis.com/css2?family=Inter:wght@300;400;500;600;700&family=Playfair+Display:wght@400;500;600;700&display=swap" rel="stylesheet">

Replace with your preferred Google Fonts.

3. Adjusting Text Sizes

To change main heading size:

<h1 class="text-[38px] md:text-6xl lg:text-7xl xl:text-[80px] font-bold">

Change values like "text-6xl" to "text-5xl" for smaller size.

üîßTroubleshooting Guide

‚ùå Error: "npm command not found"

Solution: Install Node.js first from nodejs.org

‚ùå Error: "Port 5173 is already in use"

Solution: Close other applications using that port, or use a different port by adding --port 3000

‚ùå Website doesn't appear in browser

Solution: Ensure server is running correctly, check console for error messages

‚ùå Images don't appear

Solution: Check image paths, ensure files exist in correct folders

‚ùå Changes not visible

Solution: Refresh browser (Ctrl+F5), or restart development server

üí° Debugging Tips:
  • Use F12 in browser to view errors in Console
  • Check Network tab for file loading issues
  • Ensure all files are saved properly
